# 无界UI - 桌面环境

云流操作系统的现代化桌面环境组件

## 架构概述

无界UI桌面环境采用模块化设计，包含以下核心组件：

- **桌面图标管理系统** - 图标管理、拖拽操作、状态持久化
- **任务栏组件** - 应用启动器、窗口切换、系统托盘
- **窗口管理器** - 多窗口管理、虚拟桌面、窗口特效
- **主题系统** - 视觉定制、主题管理

## 技术栈

- **前端框架**: Qt 6.0+
- **图形库**: OpenGL ES 3.0+
- **窗口系统**: Wayland/X11兼容
- **编程语言**: C++17, QML
- **构建系统**: CMake

## 项目结构

```
无界UI/
├── desktop/                 # 桌面图标管理系统
├── taskbar/                 # 任务栏组件
├── window-manager/          # 窗口管理器
├── theme-system/            # 主题系统
├── common/                  # 公共组件和工具

└── tests/                   # 测试文件
```
